# Day 1 Notes — Terminal & Git

## Five terminal commands
1. `pwd` — shows the folder I'm currently in.
2. `ls -la` — lists all files, including hidden ones.
3. `cd folder` — moves into a folder.
4. `mkdir name` — creates a new folder.
5. `cp a b` — copies a file from a to b.

## Three Git commands
1. `git status` — shows what's changed, staged, or untracked.
2. `git add .` — stages my changes so they're ready to commit.
3. `git commit -m "message"` — saves a snapshot of staged changes.
